Deep Learning for 3D reconstruction

I have presented this project as part of my Master thesis work. The goal was about deploying Deep Learning for 3D reconstruction starting from monocular video sequences. The obtained results show that this innovative approach can overcome some limitations provided by older algorithmic techniques. The whole project has been written in Python, exploiting Computer Vision and Deep Learning libraries, such as PyTorch, PyTorch lightining, OpenCV and Open3D.

  • Computer Vision, Deep Learning
  • Python
  • PyTorch, OpenCV, Numpy, MatplotLib, Open3D

SfM and MVS for 3D reconstruction

This project defines the baseline for my Master thesis project regarding 3D reconstruction starting from monocular video sequences. It exploits some well-known Computer Vision algorithms such as SfM (Structure from Motion) and MVS (Multi-View Stereo). The whole project has been written in Python, exploiting some open source libraries like Colmap and OpenMVS.

Augmented Reality from Scratch

This is an academic project which exploits Computer Vision algorithms in order to superimpose an Augmented Reality image over the frame of a video sequence. The whole project has been written in Python, using the Jupyter Notebook as IDE. The main algorithm is based on the extraction of SIFT descriptors and the computation of homography transform matrices.

  • Computer Vision, Augmented Reality
  • Python
  • OpenCV, Numpy
